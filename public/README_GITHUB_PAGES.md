# Despliegue en GitHub Pages

La version 1 de la landing esta lista en la carpeta `docs/`.

## Pasos

1. Sube este proyecto a un repositorio de GitHub.
2. En GitHub entra a `Settings`.
3. Ve a `Pages`.
4. En `Build and deployment`, selecciona `Deploy from a branch`.
5. En `Branch`, elige `main` y la carpeta `/docs`.
6. Guarda los cambios.

GitHub Pages publicara `docs/index.html` como pagina principal.

## Archivos incluidos

- `docs/index.html`
- `docs/assets/styles.css`
- `docs/assets/main.js`
- `docs/assets/hero-sums-system.png`
- `docs/assets/imss-bienestar-logo.png`
- PDFs enlazados desde la landing

La carpeta `aas/` y `aas.zip` quedaron ignoradas para evitar subir archivos auxiliares.
